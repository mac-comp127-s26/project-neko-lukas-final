Title: Particle Simulation
Authors: Lukas Wilson, Neko Henderson

Summary: Not a particle simulation, but a simulation of bouncing balls that adhere to gravity.

Requirements:
    Other than Java, nothing more!

To use, simply run the main method and click on the window! Clicking on the window will spawn a ball that will be given an initial velocity. Click more than once and enjoy watching them bounce around and interacting with each other.

Reference Acknowledgements:
    Homework References:
        - Breakout served as a starting point to be adapted into a new application.
        - In the Bubbles homework, cannon ball was used as reference when it came to incorporating gravity.

Known issues:
    - Kilt graphics was not made optimized for this usage, so performance is poor. The program runs well when there are 0 to 14 active balls, but performance begins to somewhat decline from 15 to 20, and severely declines beyond 20.
    - When a ball should come to a stop, it still jitters.
    - When a ball comes to a resting state on top of another ball, it does not "roll" off even when off center.
    - Balls bounce off of each other in a similar manner as bouncy permanently orientated cubes.
    - When a ball bounces off of another ball, each ball acts as though it has bounced against an immovable object.
    - Balls do not tranfer any energy to other balls. Ex. when a moving ball hits a static ball, the moving ball will bounce but the static ball will remain in position.

Societal Impact:
    This project was not written with accessibility in mind. However, its simplicity lends itself to being relatively accessible. Its largest discriminating limitations is the requirement of sight and requiring the use of a mouse. As an entirely visual application, someone without sight will not be able to use the application, and there is no audio description feature. Other neccessities are access to a pc and internet, which not everyone is fortunte enough to have. Otherwise, there are no possible unintended consquences or malicious/oppressive situations.
